# 🎓 AMC Web Platform

Interface web moderne pour créer et gérer des QCM avec Auto Multiple Choice.

## ✨ Fonctionnalités

- 🔐 Authentification multi-utilisateurs
- 📝 Création de QCM intuitifs  
- 🎯 Génération LaTeX automatique
- 📊 Dashboard personnel
- 📱 Interface responsive

## 🚀 Installation

```bash
git clone https://github.com/VOTRE-USERNAME/amc-web-platform.git
cd amc-web-platform
sudo apt install apache2 php php-sqlite3 auto-multiple-choice
